# Infinite Runner Factory 素材アセット一覧

後から手描き画像や自作BGMへ差し替える想定の必要素材です。現バージョンは Canvas の図形で仮表示しています。

デバッグ版の「ASSET SCENE」では、このファイルの項目名を選ぶと、その素材が使われる具体的なゲーム場面を静止状態で確認できます。項目を追加すると選択肢にも自動で反映されます。

### 最終管理機の制御プロトコル攻撃

画像未実装時の基準色です。色そのものに性格や感情の意味は持たせず、旧生産網の部材・警告灯・検査装置の色として扱います。

- 基準走査: くすんだ警告黄 `#d7a44a`
- 地盤校正: 土色 `#9a7652`
- 冷媒循環検査: 冷媒識別ピンク `#c96f86`
- 極性切替: 酸化銅色 `#2e9b8f`
- 搬送レーン制御: 黄鉄色 `#c9a227`
- 偏向板制御: くすんだ紫灰 `#7668a9`
- パルス砲接続: 青灰 `#9ab8c2`
- 残像中継: 銅色 `#d4a06f`
- 記録再生機構: 無彩色の機械灰 `#b8b1a3`
